# 아이맘가이드 네이버 SEO 최적화 가이드

## 1. 네이버 서치어드바이저 등록

### 1.1 사이트 등록
1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
2. 웹마스터 도구 → 사이트 관리 → 사이트 등록
3. `https://imomguide.pages.dev` 입력
4. 소유 확인 방법 선택:
   - **HTML 태그 방식 (권장)**: 메타 태그를 index.html에 추가
   
### 1.2 소유 확인 메타 태그
```html
<meta name="naver-site-verification" content="YOUR_VERIFICATION_CODE" />
```

### 1.3 sitemap.xml 제출
사이트 등록 후 → 요청 → 사이트맵 제출
- URL: `https://imomguide.pages.dev/sitemap.xml`

### 1.4 RSS 피드 제출
요청 → RSS 제출
- URL: `https://imomguide.pages.dev/rss.xml`

## 3. 네이버 SEO 체크리스트

### 3.1 기술적 SEO
- [ ] 모바일 최적화 (반응형)
- [ ] 페이지 로딩 속도 3초 이내
- [ ] HTTPS 적용 (Cloudflare 자동)
- [ ] sitemap.xml 제출
- [ ] robots.txt 설정
- [ ] 구조화된 데이터 (JSON-LD)

### 3.2 콘텐츠 SEO
- [ ] 타이틀 태그 30자 이내, 키워드 포함
- [ ] 메타 디스크립션 80자 이내
- [ ] H1 태그 페이지당 1개
- [ ] H2, H3 계층 구조 준수
- [ ] 이미지 alt 태그 필수
- [ ] 본문 2,500자 이상 (가이드 콘텐츠)
- [ ] 키워드 자연스럽게 7-10회 포함

### 3.3 네이버 특화 최적화
- [ ] 네이버 블로그 연동 포스팅
- [ ] 네이버 지식iN 활용
- [ ] 키워드 도구로 검색량 확인
- [ ] 모바일 검색 결과 미리보기 확인

## 4. 키워드 전략

### 4.1 메인 키워드 (경쟁 높음)
- 임신 초기 증상
- 출산예정일 계산
- 이유식 시작 시기
- 아기 예방접종

### 4.2 롱테일 키워드 (추천)
- 임신 8주차 입덧 심할때
- 생후 6개월 아기 수유량
- 신생아 밤잠 안잘때
- 이유식 초기 재료 순서

### 4.3 월별 계절 키워드
- 1-2월: 겨울 신생아 외출, 감기 예방
- 3-4월: 봄 아기 알레르기, 외출 준비
- 5-6월: 여름 기저귀 발진, 땀띠
- 9-10월: 가을 이유식, 환절기 건강
- 11-12월: 겨울 방한용품, 크리스마스 선물
